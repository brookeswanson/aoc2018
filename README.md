## To run locally
1. Place your inputs in the resources folder with the name `input-day-N.txt` (where N is the given days input)
2. Run `lein run` in the project root directory with an optional `dayN` argument.

```
$ lein run day1
=>
----- Day 1 ------------------------
Part1:
427
Part2:
341
```
